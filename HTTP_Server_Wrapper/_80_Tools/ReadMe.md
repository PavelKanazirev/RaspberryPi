# HTTP

Author: Pavel Kanazirev

[![GitHub Location](https://icon-library.net/images/github-icon-png/github-icon-png-26.jpg)](https://github.com/PavelKanazirev/HTTP)

HTTP is a project representing a solution for a specific problem.
This folder does contain a set of tools that will not become part of the project deliverables but may be used for such.
These tools are identified in the list below :

  - doxygen - used for document generation
  - GNU make - used for GNU build command wrappers
  - CPPCheck - used to perform static code analysis
  - gtest - used for unit testing
  - valgrind - used for dynamic code analysis
  - ServerConfiguration - used for configuration
  - spdLog - used for logging during debug
  - Expected - used as an optimized substitution of exceptions
  - CLang - sanitizer

In order to use the tools located here - the following pre-requisites should be available:
